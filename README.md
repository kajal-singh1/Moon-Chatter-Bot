# Moon-Chatter-Bot

## 📌 Description
This is an AI-based chatbot developed using Flask and NLP techniques. 
It helps users get information about college such as courses, fees, timings, and facilities.

## ⚙️ Tech Stack
- Python
- Flask
- NLTK
- TensorFlow / Keras

## 🚀 Features
- Answer college-related queries
- NLP-based intent recognition
- Web interface using Flask

## ▶️ How to Run
1. Install dependencies
2. Run training.py
3. Run app.py
